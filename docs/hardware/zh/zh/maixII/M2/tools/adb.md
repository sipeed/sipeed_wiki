# adb 操作介绍



ADB.exe 在 MaixPy3 IDE 中的 maixpy3_examples 和 PhoenixSuit 文件夹下

## 链接开发板
将开发板上的 OTG 接口接上电脑， 然后再命令行中输入 adb shell，会自动进入到开发板的 linux 系统中

```shell
BusyBox v1.27.2 () built-in shell (ash)

------run profile file-----

root@sipeed:/#
```

这时可以开始使用 MaixPy3 进行开发了
