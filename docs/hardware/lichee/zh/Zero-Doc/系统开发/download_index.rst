Zero系统烧录
===================================

荔枝派Zero的系统烧录主要分以下几种：

1. dd镜像烧录

2. 分区镜像烧录

3. SPI Flash系统烧录

4. 另为方便广大用户使用，有Win32DiskImage一键烧录方式

Zero能够烧录的系统目前有Linux，Camdroid，Openwrt等，甚至能够运行去掉图形界面的低版本安卓，今后随着广大荔枝派爱好者开发，将会有更多可供烧录的系统面世。

如果您对开发或移植荔枝派Zero系统感兴趣，欢迎加入荔枝派DIY交流群：573832310
