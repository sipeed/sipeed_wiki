## 什么是 Maixduino

让 Maix 系列的开发板（k210 芯片）支持 Arduino IDE及库， 方便使用大量已有的开源 Arduino 库， 从而快速进行开发和原型验证

## 如何开发

Maixduino支持原始 [Arduino IDE](./get_started/install.md) 的开发，同时也可以选择支持一键调试功能的 [Platform IDE](./get_started/install.md) 开发。
